User Management System with Angular and an API Backend
1(a). Frontend (Angular): - Create a single page application with multiple components.
- Show a list of users with search and filter options.
- Add a form to create and edit users.
- Validate the form (e.g., required fields, valid email).
- Pagination and Sorting is required.

1(b). Backend: - Create an API for managing users (add, view, update, delete). Delete means Inactive.
- Each user should have: ID, Name, Email, Role (dropdown), and Status (Active/Inactive).
- Validate fields (e.g., Name, Role is required, Email must be valid).

2.Smart Grid Smart Grid
1. Grid Setup: ◦ Create a grid that consists of 9 boxes. ◦ Each box should have: ▪ A white background by default. ▪ A subtle border around it. 2. Box Behavior on Click: ◦ When a box is clicked for the first time: ▪ Change its background color from white to black. ▪ The boxes can be clicked in any order. ◦ If a box is already black: ▪ Clicking it again should have no effect (the color remains black). 3. Reset Button: ◦ Place a "Reset" button below the grid. ◦ When the Reset button is clicked: ▪ The boxes that have turned black should change back to white. ▪ The color change should happen gradually (i.e., with a visible animation). ▪ Important: The boxes should reset in the exact same order in which they were originally clicked (i.e., the order in which they turned black).
