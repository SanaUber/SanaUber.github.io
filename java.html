User Management System with Angular and an API Backend
1. Backend: - Create an API for managing users (add, view, update, delete). Delete means Inactive.
- Each user should have: ID, Name, Email, Role (dropdown), and Status (Active/Inactive).
- Validate fields (e.g., Name, Role is required, Email must be valid).



2. Frontend (Angular): - Create a single page application with multiple components.
- Show a list of users with search and filter options.
- Add a form to create and edit users.
- Validate the form (e.g., required fields, valid email).
- Pagination and Sorting is required.
