JDBC-based core Java console application

Console Application
JDBC-based core Java console application designed for basic CRUD operations on a MySQL database.
JDBC Driver: The application uses the com.mysql.cj.jdbc.Driver to establish a connection with the MySQL database.
It enables Java programs to communicate with MySQL.
Prepared Statements: Prepared statements are used for secure and efficient query execution.
They support parameterized queries for operations like INSERT, DELETE, and UPDATE.
Query Execution: Queries are executed using methods like executeUpdate for modifying data and executeQuery
for fetching results from the database.
Result Handling: The application retrieves and processes data using a ResultSet object,
iterating through rows to display database records.
Dependencies Management: The project relies on the mysql-connector-java-8.0.26.jar library to enable JDBC functionality
and manage database connections effectively.
